# NetFlow Analysis Dashboard

A lightweight network traffic monitoring and NetFlow analysis dashboard built using Node.js and JavaScript.

## Features
- Network flow visualization
- Traffic monitoring dashboard
- Backend API integration
- Interactive frontend interface

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js

## Project Files
```text
index.html
script.js
style.css
server.js
package.json
```

## Installation
```bash
npm install
npm start
```

## Purpose
This project was created to explore network traffic analytics and visualization concepts using modern web technologies.
